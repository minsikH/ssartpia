    오시는 길

    • 오시는 길

      • 대중 교통

        • 버스 204, 403, 449, 814
        • 지하철 지하철 3호선 수성못역 하차, 도보 11분, 택시 기본요금

      • 철도 이용

        하차지점 오시는 길
        동대구역 하차

        대중교통 이용 시

        • 814(동대구역 건너 승차) → 수성아트피아앞 하차 1호선 동대구역 승차 → 명덕역에서 3호선 환승 → 3호선 수성못역 하차 → 도보 11분

        택시 이용 시

        • 동대구역 앞 택시정거장 이용, 17분 소요
        대구역 하차

        대중교통 이용 시

        • 204(경상감영공원앞 승차) → 수성아트피아앞 하차 1호선 대구역 승차 → 명덕역 3호선 환승 → 3호선 수성못역 하차 → 도보 11분

        택시 이용 시

        • 롯데백화점 정문에서 택시정거장 이용, 20분 소요
        서대구역 하차

        대중교통 이용 시

        • 204(서대구역 남측1)승차 → 수성아트피아앞 하차

        택시 이용 시

        • 서대구역 앞 택시정거장 이용, 27분 소요

      • 고속도로 이용

        IC 오시는길
        수성 IC

        수성 IC

        • 수성IC에서 ‘대구국제공항, 경산’ 방면으로 우회전 → 월드컵삼거리에서 ‘대구국제공항’ 방면으로 좌회전 → ‘황금네거리’ 방면으로 오른쪽 방향 → 황금고가교에서 ‘중동교, 황금네거리’ 방면으로 왼쪽 고가차도 진입 → 황금네거리에서 ‘수성못, TBC’ 방면으로 좌회전 → 두산오거리에서 ‘대구지방경찰청, 수성아트피아’ 방면으로 좌회전 → 수성아트피아 주차장으로 진입
        동대구 IC

        동대구 IC

        • 동대구IC에서 ‘시청’방면으로 왼쪽 고가차도 진입 → 망우당고가차도에서 ‘시청, 동대구역복합환승센터’ 방면으로 왼쪽 고가차도 진입 → 효목네거리에서 ‘국립대구박물관, 만촌네거리, 무열대’ 방면으로 좌회전 → 무학네거리에서 ‘수성아트피아, 대구경찰청’ 방면으로 우회전 → ‘지범로 17길’ 방면으로 좌회전 → 수성아트피아 주차장으로 진입
        서대구역 IC

        서대구역 IC

        • 남대구톨게이트에서 ‘본리네거리’ 방면으로 오른쪽 방향 → 대명역에서 ‘대명로14길’ 방면으로 우회전 → 700m 직진 후 좌회전 → 대명교에서 ‘신천대로’ 방면으로 좌회전 → 안지랑고가교에서 ‘가창, 신천대로, 남구국제스포츠클라이밍장’ 방면으로 왼쪽 고가차도 진입 → 현충고가교에서 ‘가창, 신천대로’ 방면으로 고가차도 진입 → 앞산고가교에서 ‘대구경찰청, 수성못, 남구국제스포츠클라이밍장’ 방면으로 왼쪽 고가차도 진입 → ‘앞산터널로(유료도로)’ 방면으로 우회전 → ‘대구경찰청, 두산오거리’ 방면으로 왼쪽 방향 → 수성아트피아 주차장으로 진입
        북대구 IC

        북대구 IC

        • 북대구 IC 북대구IC에서 ‘시청, 신천대로’ 방면으로 오른쪽 방향 → ‘대구국제공항, 대구스타디움, 시청’ 방면으로 도시고속도로 진입 → 성북교지하차도에서 ‘청도, 가창, 앞산순환로’ 방면으로 왼쪽 지하차도 진입 → ‘한국장학재단(KOCAF), 시청별관(구‧경북도청)’ 방면으로 좌회전 → 시청별관에서 우회전 → 신천동로진입부에서 ‘가창, 수성교’ 방면으로 오른쪽 방향 → 수성아트피아 주차장 진입
        서대구 IC

        서대구 IC

        • 서대구 IC 서대구IC에서 ‘신천대로, 서대구공단’ 방면으로 오른쪽 방향 → ‘팔달교’ 방면으로 좌회전 → 고가차도 진입 → 도시고속도로 진입 → 매천대교지하차도에서 ‘앞산순환로, 대구국제공항, 안동’ 방면으로 왼쪽 지하차도 진입 → 팔달교지하차도에서 ‘앞산순환로, 침산교’ 방면으로 왼쪽 지하차도 진입 → 침산교지하차도에서 ‘앞산순환로, 시청’ 방면으로 왼쪽 지하차도 진입 → 성북교지하차도에서 ‘청도, 가창, 앞산순환로’ 방면으로 왼쪽 지하차도 진입 → ‘한국장학재단(KOSAF), 시청별관(구․경북도청)’ 방면으로 좌회전 → 시청별관에서 우회전 → 신천동로진입부에서 ‘가창, 수성교’ 방면으로 오른쪽 방향 → ‘경산, 범물’ 방면으로 좌회전 → 수성아트피아 주차장 진입
